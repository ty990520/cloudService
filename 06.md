# cd명령어 사용

### su와 sudo 명령어 차이
* sudo : 우분투유저에서 일시적으로 super user의 권한을 빌릴 때
* su : 아예 super user로 접속하는 것

<br><br>

### 상대경로
* / : 최상위 디렉토리
* ./ or . : 현재 디렉토리
* ../ : 현재 위치 상위 디렉토리
* ~/ : home디렉토리 위치

<br><br>

### 셋은 같은 경로를 가리킴
* ```~$ cd /home/ubuntu/dir1``` : 절대경로
* ```~$ cd ./dir1``` : 현재 경로
* ```~$ cd ~/dir1``` : 상대 경로
